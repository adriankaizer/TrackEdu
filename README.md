# TrackEdu
Project Overview
A clickable Figma prototype for an automated attendance system designed to support UN Sustainable Development Goal 4: Quality Education. The system integrates RFID technology simulation with a teacher dashboard to streamline attendance tracking, reduce administrative burden, and enable early intervention for at-risk students.

SDG Alignment
Target 4.1: Ensure all girls and boys complete free, equitable and quality primary and secondary education.
By automating attendance tracking, teachers save 5-10 minutes per class for instruction. Real-time data enables identification of at-risk students before they fall behind.
